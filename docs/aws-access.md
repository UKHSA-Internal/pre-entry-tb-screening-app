# Request Access to AWS Environments

- Go to the ServiceNow site https://healthprotection.service-now.com/ukhsasp?id=ukhsa_index

- In the "How can we help?" box search for User Account Maintenance & click the top link:
- In the form enter the following

  - Type of service required?
  - Request Account
  - What domain does the account belong to?
    - UKHSA
  - Please provide details of what is required:

    - I need the following roles under the Pre-entry TB Screening workload.

      AWS Develop ApplicationDeveloper,

      AWS test ApplicationDeveloper,

      AWS Preproduction ApplicationDeveloper,

      AWS Test Administrator,

      AWS Preproduction Administrator

      AWS Develop Administrator

- You should get an email confirming that the request has been submitted, and you should be able to see it if you click "My Tickets" in the top right
- Ensure you speak to Sebastian Weavers for swifter approval.
- Once you have access, go to the login at https://halopr.awsapps.com/start/#/?tab=accounts (you should be prompted to sign in with SSO if you aren't already)
